# Welcome to Pita Documentation

**Pita** (Post-training Inference Time Algorithms) is a library designed to enable probabilistic inference time algorithms for any model. It focuses on training-free reasoning and advanced sampling techniques.

## Key Features

- **Training-Free Reasoning**: Enhance model performance without additional training.
- **Multiple Backends**: Support for vLLM, DeepSpeed, Llama.cpp, and more.
- **Advanced Sampling**: Implementations of Power Sampling, Self-Consistency, and other strategies.
